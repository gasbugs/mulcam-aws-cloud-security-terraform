# Macie Sensitive Data Discovery

FA01HC 5일차 AWS 보안 서비스 이해 단원 중 "Macie를 활용한 민감 정보 탐지" 실습입니다.

## 배포 자원

- Amazon Macie 계정 활성화
- 민감 정보 탐지용 S3 샘플 버킷
- Macie one-time classification job

## 확인 포인트

1. 샘플 버킷에 테스트 객체를 업로드합니다.
2. Macie Classification Job 결과와 Finding 생성을 확인합니다.
3. S3 버킷 보안 설정과 Macie 탐지 결과를 함께 해석합니다.
