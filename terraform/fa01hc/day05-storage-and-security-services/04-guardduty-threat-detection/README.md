# GuardDuty Threat Detection

FA01HC 5일차 AWS 보안 서비스 이해 단원 중 "GuardDuty를 활용한 악성코드 탐지" 실습입니다.

## 배포 자원

- Amazon GuardDuty Detector
- EBS Malware Protection 기능

## 확인 포인트

1. GuardDuty 콘솔에서 Detector가 활성화되었는지 확인합니다.
2. Sample Findings를 생성하고 Finding 유형, Severity, Resource 정보를 읽습니다.
3. EBS Malware Protection 기능이 어떤 이벤트에서 동작하는지 설명합니다.
